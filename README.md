# 모먹(Momuk)

**Online site**: https://seonghui.github.io/momuk/

## About

* 오늘 모 먹지?의 모먹입니다.
* 조건에 맞는 음식점을 추천해 주는 간단한 App 입니다.
* 매번 점심메뉴 고르기 귀찮아서 만들었습니다.
* 제가 다니는 회사 근처 데이터만 등록되어 있습니다.

## Dependencies

* React 16.8.6
* react-modal 3.10.1
* node-sass 4.12.0
* material-ui 4.3.3"

## Running on localhost

```bash
# install dependencies
$ npm install

# 개발 서버 시작
$ npm run start
```
